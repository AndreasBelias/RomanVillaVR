# Asset Attribution

The source code, scene assembly, lighting setup, interaction logic,
3D scan integration, and optimization work were created by Andreas Belias.

assets collected during the original university assignment.

the rights and licenses of its original creators.

This repository is presented as an educational and portfolio project.

If you are the creator or rights holder of any included asset and would like
the attribution corrected or the asset removed, please contact the repository
